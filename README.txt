Real Membership is a lite plugin for managing plans and memberships.

== Description ==

Real Membership is a simple lite Membership management plugin.
Currently it supports plans and user memberships, but we plan to introduce many more features in the future.

Please note that the Real Membership plugin is still a rough beta.
So we will appreciate if you don't rate us down at the current stage.

== Installation ==

1. Upload the real-membership.zip archive to the /wp-content/plugins/ folder
2. Unzip the archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy :)
